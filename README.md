# 🎯 Guess Number Game

A Python-based command-line number guessing game with OOP design.

---

## 📌 Features

- Multiple difficulty levels (Easy / Normal / Hard)
- Random number generation
- Limited attempts system
- Game history tracking
- Win/Loss statistics
- File-based record storage

---

## 🧠 Tech Stack

- Python 3
- Object-Oriented Programming (OOP)
- File I/O

---

## 📂 Project Structure

- main.py → program entry
- game.py → game logic
- record.py → data model
- record_manager.py → data processing
- stats.py → statistics calculation
- utils.py → helper functions
- config.py → difficulty settings

---

## ▶️ How to Run

```bash
python main.py
